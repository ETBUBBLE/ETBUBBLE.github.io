  git clone wangzhi.com --克隆对应网址仓库

 git pull –更新仓库

 git add -A 添加到缓冲区，

 git commit -m“your name” 更新本地仓 

 git push 提交上网络仓库

 上面3个连着用就行了。

 git config --global user.name "输入你的用户名"

 git config --global user.email "输入你的邮箱"

 初始化下身份照着打就行了
